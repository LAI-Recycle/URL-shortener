#Getting Start
---
- Clone the project
```
$ git clone https://github.com/LAI-Recycle/URL-shortener.git
```
- Install the required dependencies
```
$ npm install
```
- Install nodemon
```
$ npm i nodemon
```
- Start the server
```
$ npm run dev
```
- Execute successfully if seeing following message
```
App is running on http://localhost:3000
```
